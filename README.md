# نظام إدارة محل الموبايلات (Mobile Shop Manager)

نظام متكامل لإدارة محلات الموبايلات، تم بناؤه باستخدام **Flutter** و **SQLite**.

## المميزات
- **نظام تسجيل الدخول والصلاحيات**: يدعم المدير والموظفين بصلاحيات مرنة.
- **إدارة المخزون**: إضافة المنتجات، الأقسام، طباعة الباركود، وتنبيهات نفاذ الكمية.
- **المبيعات**: شاشة بيع تدعم الماسح الضوئي، البيع الآجل (مديونية)، وطباعة الفواتير.
- **العملاء والموردين**: تتبع الديون والمدفوعات.
- **الصيانة**: إدارة أجهزة العملاء في الصيانة وتتبع حالتها.
- **المحافظ والدرج**: إدارة التدفقات النقدية والمحافظ الإلكترونية.
- **التقارير**: تقارير شاملة للأرباح والمبيعات والمخزون.

## المتطلبات
- Flutter SDK
- Android SDK (لبناء نسخة APK)
- Visual Studio (لبناء نسخة Windows EXE)

## كيفية التشغيل
1. قم بتحميل المستودع.
2. قم بتشغيل الأمر `flutter pub get` لتحميل المكتبات.
3. قم بتشغيل التطبيق باستخدام `flutter run`.

## الإنتاج (Build)
لإنتاج نسخة APK:
```bash
flutter build apk --release
```
لإنتاج نسخة EXE:
```bash
flutter build windows --release
```

## GitHub Actions
تم توفير ملف العمل `.github/workflows/build.yml` في الكود (يجب تفعيله يدوياً إذا لم يتم رفعه تلقائياً بسبب صلاحيات الأمان) لبناء النسخ تلقائياً عند كل دفع للكود.
